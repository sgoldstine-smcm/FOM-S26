    Foundations of Mathematics
    🔗
    First, let’s look at the formal proof that when \(x\in {\bf Z}\text{,}\) \(x\) is even if and only if \(x\) is not odd. First, a reminder of the Division Algorithm (misnamed, because it’s a theorem—or if you’re in a rush, an axiom—and not an algorithm):
    🔗
    Division Algorithm. Suppose \(a\in{\bf Z}\) and \(b\in{\bf Z}^+\text{.}\) Then there exist unique integers \(q\) and \(r\) such that \(a = bq + r\) and \(0\leq r < b\text{.}\)
    🔗
    Proposition. Suppose \(x\in{\bf Z}\text{.}\) Then \(x\) is even or \(x\) is odd, and \(x\) is not both even and odd.
    🔗
    Proof. Let \(x\) be an integer. By the Division Algorithm, there exist unique integers \(q\) and \(r\) such that \(x = 2q + r\) and \(0\leq r < 2\text{.}\) Note that this means that either \(r = 0\) or \(r = 1\text{.}\)
    🔗
    Case 1: r = 0.
    🔗
    In this case, \(x = 2q + 0 = 2q\) and \(q\) is an integer, so by definition \(x\) is even. Moreover, if \(x\) were odd, then there would exist an integer \(k\) such that \(x = 2k + 1\text{,}\) which would mean that \(r = 0\) is not unique. Thus, \(x\) is not odd.
    🔗
    Case 2: r = 1.
    🔗
    In this case, \(x = 2q + 1\) and \(q\) is an integer, so by definition \(x\) is odd. Moreover, if \(x\) were even, then there would exist an integer \(k\) such that \(x = 2k = 2k + 0\text{,}\) which would mean that \(r = 1\) is not unique. Thus, \(x\) is not even.
    🔗
    In both possible cases, \(x\) is even or odd but not both. \(\square\)
    🔗
    So, just to reaffirm: you may freely use the fact that when \(x\) is an integer, if \(x\) is not even then \(x\) is odd, and if \(x\) is not odd than \(x\) is even.
    🔗
    Now, some proofs for you to try. Some of them will use the following definition (implicit in our description of \({\bf Q}\) in Chapter 1):
    🔗
    Definition. Suppose \(x\in{\bf R}\text{.}\) Then \(x\) is rational when there exist \(a,b\in{\bf Z}\) such that \(x = a/b\text{.}\) Otherwise, \(x\) is irrational.
    🔗
    1. Suppose \(a,b,c\in{\bf Z}\text{.}\) Prove that if \(c|a\) and \(c|b\text{,}\) then \(c^2|(a^2 - ab + b^2)\text{.}\)
      🔗
      🔗
    2. Suppose that \(x,y\in{\bf R}\text{.}\)
      🔗
      Prove that if \(x\) and \(y\) are rational, then \(x + y\) is also rational.
      🔗
      🔗
    3. Suppose that \(x,y\in{\bf R}\) and that \(x\) is rational. (In other words, \(x\in{\bf Q}\text{.}\))
      🔗
      Prove that if \(xy\) is irrational, then \(y\) is irrational.
      🔗
      🔗
    4. Suppose \(n\in{\bf Z}\text{.}\) Prove that if \(n^2\) is not divisible by \(4\text{,}\) then \(n\) is odd.
      🔗
      🔗
    5. Suppose that \(k\) and \(m\) are integers. Prove that if \(k + m\) is even, then \(k - m\) is even.
      🔗
      🔗
