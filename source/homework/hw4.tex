Homework 4\\
Due Friday, April 3, in class
\end{center}


\begin{enumerate}

\item  Suppose $a\in\Z$.  Prove that if $a^2 | a$, then $a\in \{-1,0,1\}$.

\item  Suppose that $a,b\in\Z$.  Prove that $10a - 15b\neq 3$.

\item  Prove that for any $a,b\in\Z$, $(a + b)^3\equiv a^3 + b^3\ (\mod 3)$.

\item  Suppose~$x$ and~$y$ are real numbers.  

\begin{enumerate} 
\item Prove that if $(x+y)^3 = x^3 + y^3$, then $x=0$, $y=0$, or $y = -x$.

\item Prove the converse, that if $x=0$, $y=0$, or $y = -x$, then $(x+y)^3 = x^3 + y^3$.
\end{enumerate}

\textit{Remark.}  Together, the two parts of this question prove that 
$(x+y)^3 = x^3 + y^3$ if and only if $x=0$, $y=0$, or $y = -x$.

\textit{Hint.}  There is a similar odd exercise in Chapter 7.


\end{enumerate}
